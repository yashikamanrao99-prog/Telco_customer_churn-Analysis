# Telco Customer Churn Analysis

This project analyzes telecom customer data to identify churn patterns and predict customers who are likely to leave the service.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Model Building
- Model Evaluation
- Customer Segmentation

## Outcome
Developed a machine learning model to predict customer churn and gained insights into factors affecting customer retention.
